Welcome to DioxideAi

1. Install Ollama

DioxideAi connects to the local Ollama service. Download it from ollama.ai and run the installer.

2. Start the Ollama service

Keep Ollama running in the background. Launch it from Applications or run:

ollama serve

3. Pull a model

Download at least one model before chatting. Common choices:

ollama pull llama3:8b
ollama pull mistral:7b

4. Pick the right size

Match the model to your Mac's GPU/VRAM:

  • 16 GB VRAM: gemma:7b, llama3:8b
  • 24 GB VRAM: gemma3:12b
  • 32 GB VRAM: gpt-oss:20b
  • 48 GB VRAM: llama3:70b (Q4)
  • 64 GB VRAM and up: gpt-oss:70b, mixtral:8x22b (Q4/Q5)

Check VRAM via About This Mac → System Report → Graphics/Displays.

5. Connect in DioxideAi

With Ollama running, pick your downloaded model in the header and start chatting. Fresh web context is fetched automatically when needed.
